# 1. 标题
## （1） 一般标题
发文机关 + “关于” + 事由 + “的” + 文种
{% note info %}
例如：国务院关于坚决稳定市场物价的通知
{% endnote %}
## （2） 自拟标题
{% note info %}
例如：女神妈祖，天下一家
{% endnote %}
> 当写作对象是机关单位时，应当使用一般标题（正式标题）；如果是人民群众，例如村民、大学生、游客等，应当使用自拟标题。
## （3） 书写格式
标题居中书写，不带书名号

# 2. 称谓或主送机关
## （1） 称谓（对象为人物）
{% note info %}
例如：尊敬的与会嘉宾、广大的家长朋友们等
{% endnote %}
## （2） 主送机关（对象为单位）
{% note info %}
例如：各社区、各街道办等
{% endnote %}
## （3） 书写格式
顶格书写

# 3. 落款
## （1） 单位（个人）
{% note info %}
例如：某企业工会（写在日期上一行）
{% endnote %}
## （2） 日期
{% note info %}
例如：2021年XX月XX日 或 XXXX年XX月XX日
{% endnote %}
## （3） 书写格式
右下角（落款位于正文右下，日期右空四个格子，发文者以日期为准上方居中）
